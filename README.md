# Fair Dinkum Publishing: Sovereign AI for Aussie Tradies 🛠️🦘

Automate your business with the **Digital Ute**. A 14-agent AI workforce & MCP orchestrator for **Sovereign Workflows**. Handles quotes, receipts & comms with **Local AI** for data privacy. Built with Next.js 15 & Stripe for high-performance digital asset delivery. **Fair Dinkum AI.**

## 🚀 Quick Start
1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Visit `http://localhost:3000/fair-dinkum` for the landing page.

## 📦 Project Structure
- `/app/fair-dinkum`: Landing page
- `/app/fair-dinkum/store`: Ebook store with Stripe integration
- `AAW_MANIFEST.md`: The 14-agent workforce strategy
- `FAIR_DINKUM_DESIGN.md`: Original design proposal
